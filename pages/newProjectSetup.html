How to set up a new project

Step 1

Make a project folder locally on your computer.

Open the folder in your code editor, or navigate to in in your terminal

Create a new Git repository with the command git init

Step 2

If you want to publish your repository to Github, you'll need to create a repository on github.com

Add your local repo to github with the command git remote add origin {github repo URL}

Step 3

In your project you'll need a .gitignore file

This is to ensure that certain files are not tracked by Git.

Here are some common things to ignore:

node_modules .DS_Store .env

Step 4

Set up your Node Package Manager (NPM)

To do this, write the command npm init

You will get a series of questions about your project.

You can also write npm init -y to skip the questions all together!

If it worked, you should see a package.json file in your project folder

Step 5

You are now ready to install dependencies!
