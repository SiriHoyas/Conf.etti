Install and compile SASS

Step 1 - Install SASS

To install SASS you'll need to have Node Package Manager (NPM) installed.

Write the command npm install -D sass in your terminal.

SASS is now installed as a dependency in your project

Step 2 - Add the files you need

You'll need one file for your SASS, and one for your CSS (SASS compiles in to this file)

The CSS file you make is the one you link in your HTML files

Step 3 - Compile SASS in realtime

To compile the SASS when you save, go to your package.json file

You will add a script with a --watch flag.

This flag tells SASS to watch source files for changes, and re-compile to CCS each time you save your SASS

The script you are adding is "watch": "sass --watch sass-file:css-file"

Remember to change the sass-file and css-file placeholder to your output and input files!

Step 4 - Run it!

You can now run the script, and SASS compiles to CSS like magic!

In the terminal, write the command npm run watch
