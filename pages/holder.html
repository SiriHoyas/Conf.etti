Run ES Lint and Prettier on commit

Step 1 - Install ESLint and Prettier

To install ESLint and Prettier you'll need to have Node Package Manager (NPM) installed.

Use NPM to install Prettier and ESLint with the following commands

npm install --save-dev prettier npm install eslint --save-dev

Step 2 - Initialize ESLint

We need to initialize and setup rules for ESLint.

Write the command npx eslint --init and you will get a series of questions to setup ESLint

Step 3 - Install package to generate scripts

Use the tool Mrm to generate script to run Prettier and ESLint on commit.

This is a pre-commit hook

In your terminal, write npx mrm@2 lint-staged

Step 4 - Set up tasks

In your package.json file, you will now see a field "lint-staged":

The task "eslint --fix" will run ESLint on the files selected

while "prettier --write" will format your code.
